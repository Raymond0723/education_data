# MySQL Installer 下載教學

本指南將引導您如何從 MySQL 官網獲取正確的安裝檔。  
下載網址=>https://dev.mysql.com/downloads/installer/
## 📥 下載流程圖

```mermaid
graph TD
    Start([前往官網]) --> OS[選擇 Windows 系統]
    OS --> Select[找到 8.0.x MSI Installer]
    Select --> ClickDL[點擊 Download]
    ClickDL --> Skip[點擊 No thanks, just start my download]
    Skip --> Done([下載完成])
